Este código implementa una versión simplificada del comando Unix wc (word count). La clase App recibe parámetros desde la línea de comandos para contar caracteres (-c), líneas (-l) y palabras (-w) en un archivo de texto, validando errores de argumentos, de lectura y comandos no reconocidos. La clase Counter proces­a el archivo carácter por carácter y calcula los totales de caracteres, líneas y palabras, que luego son impresos según los comandos solicitados.
